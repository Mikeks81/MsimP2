Portfolio

For my final project I made an app that focuses on simplifying a freelancers workflow by automating bookeeping and billing work for them. Users can manage their clients and jobs those clients give them. Users can generate and email invoice PDF's directly from the app. All finances are then viusalized using D3.js chart and YTD figures. I built FreelanceMe using Ruby on Rails, HTML, Sass, JQuery, D3.js and AJAX. I heavily focused on the UI/UX element of the app by implementing Ajax to create an uninterrupted experience.

Freelanceme.tech

For my final group project we built an app aimed to empower victims of street harassment. Users can anonymously report and map harassment incidents. We built this app with Ruby on Rails, JavaScript, Jquery, HTML and Sass. We pulled in Google Maps and Places API to find and map locations of harassment defined by the user. We used Github as version control and pair programmed during parts of developement. Lastly we used media queries in order to make it fully mobile ready.

Street-Report.com

Boombox is a music player made using Vanilla JavaScript as an Object literal. You can stream songs with a url. The display shows the current song/album and the progress/duration of the song. The music can be controlled by clicking the tape buttons at the bottom.
